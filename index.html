1. 我们如何收集和使用您的个人信息

我们坚持最小化数据处理原则。作为一款离线工具，我们仅在您的设备本地处理必要的数据以提供功能。具体情况如下：

1.1 核心业务功能

(1) 文件编辑与管理

为了实现读取、编辑、保存、重命名、删除文件等核心功能，我们需要访问您的设备存储空间。我们仅会在您主动操作（如点击“打开文件”、“保存”）时，通过系统文件选择器或应用沙箱机制处理文件。

  • 涉及信息： 您选择的文本文件、代码文件、图片等文档内容。
  • 使用方式： 数据仅在内存中即时处理并在您的设备本地存储中读写。我们不会上传、不会备份、也不会分析您的文件内容。

(2) 剪贴板功能

为了辅助您进行文本编辑，我们需要使用系统剪贴板功能。

  • 使用场景： 仅当您主动在预览界面点击“复制链接”或在编辑器中选中文字进行复制/粘贴操作时。
  • 承诺： 我们不会后台监听您的剪贴板内容，也不会读取您在其他应用中复制的无关信息。我们仅响应您的主动写入或粘贴指令。

1.2 附加业务功能

(1) 智感握姿与界面适配

为了提供更便捷的单手操作体验（如自动判断左右手以调整侧边栏位置），以及适配华为 HDS (HarmonyOS Design System) 光效，我们使用了 HarmonyOS 的传感器能力。

  • 涉及权限： 传感器信息 (MultimodalAwarenessKit / Motion)。
  • 涉及信息： 设备的握持状态（左手握持、右手握持或双手握持）、设备类型、系统版本。
  • 使用方式： 该数据由系统 API 实时返回，仅用于即时调整 UI 布局（如侧边栏位置）。该数据不会被保存，也不会被上传。

(2) 网页预览渲染

当您使用 Markdown 或 HTML 预览功能时，我们会使用系统提供的 WebView 组件在本地渲染您的文档内容。此过程完全在本地运行，不涉及服务端转码。

1.3 申请的系统权限说明

权限名称 使用目的与场景
存储权限 用于读取、修改、保存您指定的文档文件。
剪贴板 响应您的主动操作（复制/粘贴文本或链接）。
传感器 用于“智感握姿”功能，识别左右手习惯以自动调整界面布局。

2. 我们如何存储您的个人信息

2.1 存储地点

我们承诺：本应用产生或处理的所有数据（包括您的文档、应用偏好设置、浏览记录等）均存储于您的移动终端设备本地。

我们未在任何国家或地区部署用于存储用户数据的服务器，因此不存在数据跨境传输的情况。

2.2 存储期限

  • 文档数据： 存储期限完全由您自行决定。当您在应用内或系统文件管理器中删除文件后，数据将被永久移除。
  • 应用配置： 存储于本地首选项中，当您卸载应用或“清除应用数据”时，这些配置将被立即删除。

3. 我们如何保护您的个人信息

虽然我们不收集您的信息上传至云端，但我们仍采取多重安全措施保护您本地数据的安全：

  • 本地沙箱机制： 本应用遵循 HarmonyOS 的应用沙箱机制，您的数据与其他应用隔离，防止被恶意应用非法读取。
  • 无网络权限： 在技术架构上，我们最大限度减少了网络权限的使用，从根本上切断了数据通过本应用泄露至互联网的可能性。
  • 最小化权限申请： 我们仅申请实现功能所必需的权限，并仅在您使用相关功能时请求授权。

4. 您的权利与信息管理

您对您的个人信息拥有完全的控制权，您可以便捷地进行以下操作：

4.1 访问、复制与转移

由于数据存储在您本地，您可以通过应用内的“文件管理器”或系统的“文件”应用，随时查看、复制或将您的文件转移到其他设备。

4.2 删除信息

您可以随时删除应用内的文档。对于应用产生的缓存或设置，您可以通过系统设置中的“应用管理 -> PureEditor -> 存储 -> 清除数据”来彻底删除。

4.3 改变授权范围

您可以通过设备的“设置”应用，找到本应用的权限设置，随时关闭“存储”、“传感器”等权限。请注意，关闭相关权限可能会导致部分功能（如打开文件、智感握姿）无法正常使用。

5. 未成年人保护

我们非常重视对未成年人个人信息的保护。

  • 适用性： PureEditor 是一款通用的文本处理工具，不包含任何不适宜未成年人的内容（如暴力、色情、赌博等）。
  • 监护人责任： 如果您是未滿 18 周岁的未成年人，建议您在父母或监护人的指导下阅读本政策并使用本应用。
  • 隐私保护： 由于我们不收集任何个人信息，也不具备账号注册功能，因此我们不会建立未成年人个人信息库，也不会对未成年人进行画像或推送广告。
  • 家长控制： 监护人可以通过设备系统的“健康使用手机”或“家长控制”功能，管理未成年人对本应用的使用时长。

6. 系统基础服务说明

为了保障应用的稳定运行和功能实现，我们使用了华为 HarmonyOS 提供的系统基础服务（Native SDK）。这些服务属于操作系统的一部分，不属于第三方数据收集SDK。

服务名称 提供方 使用目的 数据处理方式
HarmonyOS Motion Kit 华为软件技术有限公司 识别握持姿态（智感握姿） 数据在本地处理，不上传
ArkUI / ArkWeb 华为软件技术有限公司 构建界面与渲染网页预览 本地渲染，不上传

7. 本政策的更新

随着法律法规的出台或业务功能的调整，我们可能会修订本隐私政策。由于本应用完全离线运行，更新后的隐私政策将随应用版本更新一同发布。当您更新应用版本后，我们会在首次启动时再次提示您阅读变更后的协议。
